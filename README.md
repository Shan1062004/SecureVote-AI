# 🗳️ VoteVision AI – Smart Voting System

## 🚀 Overview
VoteVision AI is a computer vision-based voting system designed to ensure secure and fraud-free elections using facial recognition technology.

---

## ✨ Features
- 🎯 Face-based voter authentication
- 🔐 Fraud prevention (duplicate vote detection)
- ⚡ Fast and contactless voting
- 📊 Real-time vote processing

---

## 🛠 Tech Stack
- Python
- OpenCV
- Flask
- Machine Learning (LBPH)

---

## 🧠 How It Works
1. Voter face is captured
2. System verifies identity
3. If matched → vote allowed
4. Prevents duplicate voting
